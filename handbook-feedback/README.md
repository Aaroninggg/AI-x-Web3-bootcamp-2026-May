# Handbook Feedback

阅读 AI × Web3 School Handbook 各章节后的想法、感受和反馈。

## 使用方式

按 Handbook 章节或主题建文件，命名格式：

```
handbook-feedback/
├── ai-agent-chapter.md
├── prompt-chapter.md
├── web3-wallet-chapter.md
├── smart-contract-chapter.md
└── ...
```

## 建议记录的内容

- 这章讲了什么核心概念？（一句话）
- 哪些部分讲得清晰 / 哪些容易困惑？
- 有实际帮助的例子或类比？
- 对学习路径或教学方式的改进建议
- 与自己的背景知识产生了什么连接？
