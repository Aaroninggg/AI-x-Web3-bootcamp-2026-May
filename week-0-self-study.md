# AI基础概念
AI：感知，判断，生成，行动
AI的4层能力：
- 数据层：投喂给AI的数据，有文本，图片，语音，行为数据。
	- 行为数据分为，Action（用户的行为），Object(用户行为作用的对象)，Context（上下文，用户行为发生的环境。
	- 行为数据加入训练可让模型拥有预测与判断能力，
		- 机器学习 通过行为数据对于下一步进行**预测与推荐**
		- Agent 通过行为数据进行**意图识别**，理解用户发送的复杂任务描述，进行拆解与任务编排
		- Web3 场景下，分析行为数据来判断某笔交易是否存在黑客工具或异常操作。**异常检测**
- 模型层：学习数据规律并输出结果（y=f(x) 拟合）的核心系统，
	- 类似于分类模型，回归模型，生成模型，多模态模型，判别模型，强化学习模型。自监督学习模型，预测模型，嵌入模型。后两个Web3相关
- 应用层：将模型封装成产品，例如 检索功能，代码补齐，图像生成，Skills
- 执行层：跳出一问一答的回答模式，直接将回答变成行为，进行执行。OpenClaw，Claude Code

# 机器学习入门

Machine Learning 常见流程压缩成为5步：
	1. 定义任务：最终的模型要预测什么
	2. 准备数据：收集数据，数据清晰，添加标签
	3. 训练模型：模型反复调整参数，减少错误。
		- 训练模型，为了找到一个y=f(x)的函数，既不欠拟合，也不过拟合   泛化能力适中  适度拟合
		- y 就是标签  x就是特征  f就是模型
	4. 验证模型：验证模型在未见过的数据集下的表现
	5. 上线使用：接入真实场景中。

- 用**训练集**去“喂”模型（寻找 $f$）。
- 用**验证集**去“监控”模型（发现过拟合的苗头立刻停止训练）。
- 用**测试集**做“最终评估”（确认模型在现实世界是否好用）。

有监督学习：提供标签，提供特征，寻找适度拟合的模型
无监督学习：不提供标签，只提供特征，在数据中自己寻找拟合

欠拟合，模型太笨，没有找到规则，或者规则找的不准
过拟合，在验证集表现太差，对于新数据的泛化能力不强，但是在训练集数据中的泛化能力巨强

数据质量比模型名字更重要
x与y的质量决定f的能力大小，
- 标签本身就不准
- 数据分布和真实场景不一致
- 采样有偏差
- 训练数据过旧，跟不上现实变化

# 深度学习
Deep Learning 深度学习的深度概念不是抽象的，代表神经网络的层数多、表达能力更强、能够自动从复杂数据里提取更高层次的特征。

神经网络中的多层过滤器，第一层学到较浅的模式，中间层逐步组合信息，更深层形成更抽象的表示。

深度学习的能力是靠数据规模变大，算力成本下降，模型参数规模持续提示这三件事叠加后，模型不再只适合做单点任务，而开始表现出更强的沟通能力。

# 神经网络与大模型的关系
神经网络为基础结构，Deep Learning 是多层神经网络的学习方法，大模型是Deep Learning在大规模数据与算力条件下的产物

# Deep Learning的优势与代价
## 优势
- 处理复杂与非机构化数据
- 特征工程依赖低
- 在生成、识别、理解任务上表现强
## 代价
- 训练和推理成本高
- 对数据规模和质量敏感
- 可解释性相对较弱
- 错误可能更隐蔽，不容易一眼看出
