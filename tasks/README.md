# Tasks

记录 Bootcamp 期间完成的**所有任务**。

## 使用方式

每个任务一个 `.md` 文件，命名格式：

```
tasks/
├── task-001-deploy-counter-contract.md
├── task-002-create-github-repo.md
└── ...
```

## 任务模板

```markdown
# Task NNN: 任务名称

- **日期**：2026-05-XX
- **状态**：✅ 完成 / 🚧 进行中
- **关联**：Week X / daily-notes / experiments

## 任务描述
...

## 完成记录
- 步骤 1：
- 步骤 2：
- ...

## 输出/成果
- 链接、截图、交易哈希等
```
