## Week 0 — AI 基础概念（✅ 已完成）

**时段：** 课前自学

### 学习目标
- 理解 AI 的 4 层能力架构：数据层 → 模型层 → 应用层 → 执行层
- 掌握机器学习 5 步流程：数据采集 → 清洗 → 训练 → 测试 → 部署
- 区分 AI / ML / Deep Learning / 模型 / Prompt / Agent 等核心概念
- 了解行为数据在 Web3 场景的应用（异常检测、意图识别）

## Week 1 — AI Agent & Web3 基础（🚧 进行中）

**时段：** 周二 5/19 ~ 周六 5/23 17:00

### 学习目标
- **AI 侧**：理解 Agent 架构（Tool Use、Planning、State、Human-in-the-loop），掌握 Prompt Engineering 原则，了解 Context Engineering & RAG
- **Web3 侧**：掌握钱包/签名/交易流程，部署最小智能合约，理解 L1/L2/Gas 机制
