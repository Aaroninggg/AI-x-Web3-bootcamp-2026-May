# Experiments

记录所有尝试、实验及其结果 — 成功的、失败的、学到教训的都记。

## 使用方式

每个实验一个 `.md` 文件，命名格式：

```
experiments/
├── exp-001-hermes-auto-sync.md
├── exp-002-sepolia-deploy.md
└── ...
```

## 实验模板

```markdown
# Experiment NNN: 实验名称

- **日期**：2026-05-XX
- **状态**：✅ 成功 / ❌ 失败 / 🚧 进行中
- **关联**：Week X

## 目标
想验证什么？

## 方法
怎么做的？

## 结果
实际发生了什么？

## 学到的教训
下次可以怎么改进？
```
